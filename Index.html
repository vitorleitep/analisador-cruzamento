    ℹ️ Sobre o Sistema

    🎉 Sistema Desenvolvido com Sucesso!

    Analisador completo de cruzamento bovino com funcionalidades avançadas de genealogia e inbreeding.

    🔬 Funcionalidades Principais:

    • 📷 Upload de Genealogias: Sistema de reconhecimento OCR para extrair dados automaticamente
    • 🧮 Cálculo de Inbreeding: Algoritmo baseado no Coeficiente de Wright para análise precisa
    • 💾 Banco de Dados: Armazenamento local de animais para reutilização
    • 📊 Histórico Completo: Registro de todos os cruzamentos realizados
    • ⚠️ Avaliação de Risco: Sistema inteligente de recomendações genéticas
    • 📱 Responsivo: Funciona perfeitamente em desktop e mobile

    📈 Metodologia Científica:

    O sistema utiliza o Coeficiente de Inbreeding de Wright (F) para calcular a probabilidade de homozigose por descendência. A fórmula F = ∑(1/2)^(n1+n2+1) considera as distâncias genealógicas entre os pais e seus ancestrais comuns, proporcionando uma análise precisa do risco genético da progênie.

    🎯 Níveis de Risco:

    ✅ Baixo Risco (< 6.25%)
    Cruzamento recomendado com boa diversidade genética
    ⚠️ Risco Moderado (6.25% - 12.5%)
    Monitoramento recomendado, considerar outcrossing
    ❌ Alto Risco (> 12.5%)
    Avaliar alternativas, consultar geneticista
